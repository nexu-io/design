# nexu-ui

Nexu component library workspace.

## Packages

- `@nexu/tokens`: design tokens and theme CSS
- `@nexu/ui-web`: React web component library
- `@nexu/storybook`: component workbench

## Commands

```bash
pnpm install
pnpm dev
pnpm typecheck
pnpm build
pnpm build:packages
pnpm pack:check
pnpm release:check
```
